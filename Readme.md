## Демо-проект на разных JS-фреймворках

Должен содержать страницы:

- `/` - стартовая страница.
- `/login` - страница входа в систему.
- `/register` - страница регистрации пользователя.
- `/photos` - страница с изображениями, загружаемыми по запросу https://jsonplaceholder.typicode.com/photos . Ввиду большого числа изображений, должна быть пагинация.
- `/profile` - информация о пользователе (имя и id).

Сделать на фреймворках 

- Vanilla JS 
- Angular 1.x
- Angular 4 
- React, Redux
- jQuery
